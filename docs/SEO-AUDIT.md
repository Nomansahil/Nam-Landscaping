# SEO Audit — NAM Landscaping

Scope: full audit of the existing static site (20 pages) generated from `build/`.
This audit reflects the state **at the start of this pass** and the resolution applied.
Severity: Critical / High / Medium / Low / Informational.

## Architecture (as found)
- Static HTML/CSS/JS, no framework. Pages generated by a Node script (`build/build.js`)
  from a single source of truth (`build/templates.js` + content data files).
- Already in place from earlier work: keyword-first H1s, per-page meta, canonicals,
  breadcrumbs, Service/FAQ/Breadcrumb schema, sitemap.xml, robots.txt, WebP imagery,
  responsive heroes, mobile-first CSS, three-channel CTAs.
